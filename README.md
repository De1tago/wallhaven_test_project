# 🖼️ Wallhaven Desktop Viewer

**Современный и быстрый просмотрщик обоев для рабочего стола, разработанный на Python и GTK4.**

Wallhaven Desktop Viewer предоставляет удобный интерфейс для поиска, фильтрации и загрузки высококачественных обоев с популярного ресурса [wallhaven.cc](https://wallhaven.cc) без необходимости использования веб-браузера.

---

## ✨ Основные возможности

* **Скелетная загрузка (Skeleton Loading):** Используется умная многопоточная загрузка и кэширование миниатюр, что обеспечивает **плавный** и быстрый просмотр сетки без задержек.
* **Гибкий поиск:** Поиск по ключевым словам, категориям, степени чистоты (**SFW/Sketchy/NSFW**).
* **Продвинутая фильтрация:** Выбор соотношения сторон (`16:9`, `21:9` и т.д.) и минимального разрешения (вплоть до `8K`).
* **Поддержка API-ключа:** Возможность подключения вашего личного API-ключа для доступа к контенту с пометками **Sketchy** и **NSFW**.
* **Просмотр и загрузка:** Открытие полноразмерного изображения в отдельном окне с индикатором прогресса загрузки.
* **Кастомизация:** Настройка количества колонок в сетке под размер вашего экрана.

## 💻 Установка и запуск

Для запуска приложения требуется **Python 3.x** и установленные библиотеки **GTK4 / PyGObject** в вашей системе (например, `gir1.2-gtk-4.0` в Ubuntu).

### 1. Установка зависимостей Python

Перейдите в папку проекта и установите необходимые библиотеки:

```bash
pip install gi requests
````

*(**Примечание:** Для работы с PyGObject в некоторых окружениях может потребоваться `pip install pygobject`)*

### 2\. Запуск приложения

Просто запустите основной скрипт:

```bash
python3 wallhaven_viewer.py
```

## ⚙️ Настройка и использование

### 🔍 Панель поиска и фильтров

  * **Поле ввода:** Используйте для ввода ключевых слов (например, `cyberpunk`, `nature`).
  * **Категории и Чистота (Purity):** Используйте кнопки-переключатели (`General`, `Anime`, `People` и т.д.) для настройки типа контента.
  * **Сортировка (Sort):** Выберите режим сортировки: `Toplist` (Лучшее за период), `Date Added` (Недавно добавленное) или `Relevance` (Актуальность).

### 🔑 API Ключ (Для контента 18+)

Если вы хотите видеть контент с пометкой **Sketchy** или **NSFW**, вам потребуется API-ключ от Wallhaven.

1.  Зарегистрируйтесь на [wallhaven.cc].
2.  Перейдите в настройки аккаунта и скопируйте ваш API-ключ.
3.  В приложении нажмите на кнопку **Настройки** (шестеренка) в заголовке.
4.  Вставьте ключ в поле **API Ключ**.

### 📁 Папка для сохранения

В окне настроек вы можете указать **Папку для сохранения**. Если поле оставить пустым, программа будет спрашивать, куда сохранить файл, при каждой загрузке.
